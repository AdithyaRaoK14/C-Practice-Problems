To compile and run the program in Linux terminal"

1. Open the terminal
2. create a folder mkdir folder_name
3. goto folder cd folder_name
4. copy main.c day1.h and day1_operation.c to this folder
5. to compile the program: cc main.c day1_operation.c
6. to RUN: ./a.out


to create a newfile in Linux
subl main.c
or gedit main.c
or vi main.c
